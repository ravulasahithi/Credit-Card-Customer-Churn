# Customer Churn Analytics Dashboard

An end-to-end **Customer Churn Analysis** project that leverages **SQL, Python, and Power BI** to analyze customer attrition, identify churn drivers, and present actionable business insights through an interactive dashboard.

---

## Project Overview

Customer churn is a critical challenge for financial institutions, as retaining existing customers is often more cost-effective than acquiring new ones. This project analyzes a banking customer dataset to understand the demographic, behavioral, and financial factors that contribute to customer churn.

The project follows the complete data analytics lifecycle—from data validation and cleaning to exploratory analysis, SQL-based business insights, and interactive dashboard development.

---

##  Objectives

- Analyze customer churn trends.
- Identify key demographic, behavioral, and financial factors affecting churn.
- Perform data validation and cleaning.
- Generate business insights using SQL and Python.
- Build an interactive Power BI dashboard for decision-making.
- Recommend strategies to improve customer retention.

#  Key Business Insights

- Customers with lower transaction activity are more likely to churn.
- Higher inactivity months strongly correlate with customer attrition.
- Credit utilization patterns differ between retained and churned customers.
- Customer relationship count influences customer retention.
- Certain demographic segments exhibit higher churn rates.
- Customer engagement metrics can help identify high-risk customers.
